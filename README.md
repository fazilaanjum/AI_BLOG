# ✍️ AI Blog Generator with SEO Optimization

An AI-powered blog writing application that generates structured, SEO-friendly blog posts using Google Gemini.

## Features

- 📝 Generate complete blog posts
- 🎯 Audience-specific writing
- 🎨 Multiple writing tones
- 🔍 SEO keyword optimization
- 📑 Markdown-formatted output

## Technologies Used

- Python
- Google Gemini API
- Gradio

## Installation

Clone the repository:

```bash
git clone https://github.com/fazilaanjum/AI-Blog-Generator.git
cd AI-Blog-Generator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Set your API key:

```bash
export GEMINI_API_KEY=YOUR_API_KEY
```

Run:

```bash
python app.py
```
